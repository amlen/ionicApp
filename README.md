# ionicApp

###	La description du projet

##	L’objectif :

L’objectif de projet est de crée l’application hybride ECAM events.

##	A qui s’adresse le site 

Cette application est adressée à tous les étudient et le personnel de l’ECAM.

##	Fonctionnalités de l’application :

L’application comprend principalement cinq fonctions :
- La gestion des événements : en donnant toutes les informations concernant chaque événement ainsi que la possibilité de modifier ou supprimer ce dernier.

- La gestion des catégories : permet d’afficher, modifier et supprimer des catégories.
